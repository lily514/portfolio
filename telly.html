Alexa Skill: Telly

About

  • Telly is an Alexa Skill (Python) designed to help you find the best way to stream, rent or buy the movie or TV show you want to watch
  • Uses Amazon Web Services, including S3 and Lambda Functions
  • Gathers data from JustWatch API
  • Providers include Netflix, Hulu, AppleTV, Amazon and Amazon Prime
