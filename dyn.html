DynSuite

About

  • Molecular Dynamics Program (C++)
  • Uses Gaussian 09 to calculate forces acting on each atom
  • Implements a possible reaction trajectory using Verlet algorithms
  • Aggregates trajectories together
  • Uses a Fortran program, Packmol, to initialize an explicit solvent container
  • Relies on the Abstract Factory Pattern to implement different Verlet algorithms and solvent containers.
